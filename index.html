About me

A self taught freelance developer based in Toronto. I specialize in creating responsive and functional web applications using Javascript, HTML, and CSS. My coding journey started off with an interest in game development which quickly evolved into web applications. After years of discipline and dedication I have realized the possibilities with Javascript are endless. With this creative freedom along with my passion for design and a minimalist approach, I aim to give you a clean and modern application. Most recently, I have been striving to learn web 3.0 and build immersive applications onto a blockchain. Stay tuned!
